Instructions d'utilisation
Bienvenue sur Curiosity.

Cet appareil vous permet de transférer la matière entre deux moments distincts de la réalité temporelle.

Pour lancer le transfert temporel veuillez activer 3 paramètres :
I) Vecteur d’ancrage
II) Clé de résonance (code de déverrouillage)
III) Quatre récepteurs spatio-temporels

Authentifiez vous puis lancez le transfert temporel.

Enfin, recherchez l'objet dans l'espace.
